Java 第一课：Hello World

什么是 Java？

Java 是一门面向对象的编程语言，由 Sun Microsystems 于 1995 年发布。 它的核心理念是"一次编写，到处运行"（Write Once, Run Anywhere）—— 同一份 Java 代码可以在 Windows、macOS、Linux 等不同操作系统上运行，无需重新编译。

Java 目前广泛应用于企业级开发、Android 应用、大数据处理等领域。 作为入门语言，Java 语法严谨、结构清晰，非常适合打好编程基础。

JDK 是什么？

JDK（Java Development Kit）是 Java 开发工具包，包含编译器（javac）、 运行时（java）和标准类库。学习 Java 的第一步就是安装 JDK。

安装完成后，在终端输入以下命令验证是否成功：

java -version
javac -version

第一个程序：Hello World

新建一个文件 HelloWorld.java，写入以下代码：

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

代码逐行解释

  1. public class HelloWorld —— 定义一个公开的类，类名 必须和文件名完全一致（区分大小写）。
  2. public static void main(String[] args) —— 这是 Java 程序的入口方法。每个 Java 程序都从 main 方法开始执行。
  3. System.out.println("Hello, World!") —— 在控制台打印一行文字。System.out 代表标准输出， println 表示打印后换行。

编译与运行

打开终端，进入文件所在目录，依次执行：

javac HelloWorld.java    ← 编译，生成 HelloWorld.class 字节码文件
java HelloWorld          ← 运行（注意不要加 .class 后缀）

如果一切顺利，终端会输出：

Hello, World!

常见错误

  • 类名和文件名不一致 —— Java 要求公有类的名字必须和文件名完全相同。
  • 分号遗漏 —— 每条语句结尾必须有分号 ;。
  • 大小写错误 —— Java 区分大小写，String 和 string 是不同的。
  • 忘记保存文件 —— 运行前请确保已保存 .java 文件。

小结

今天我们了解了 Java 的基本概念，安装了 JDK，并成功运行了第一个 Java 程序。 System.out.println 将成为你最熟悉的 Java 语句之一。 下一课我们将学习 Java 的基本数据类型和变量。
